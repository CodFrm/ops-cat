# OpsKat Feature Promo Design

## Brand Source

Derived from the existing OpsKat README, app screenshots, and logo assets in this repository.

## Visual Identity

- Mood: precise, operational, trustworthy, product-led.
- Canvas: dark desktop UI atmosphere, with enough green, amber, and coral signal colors to avoid a one-note blue interface.
- Background: `#071016`
- Panel: `#101923`
- Panel deep: `#0B121A`
- Foreground: `#EEF4F7`
- Muted foreground: `#AAB5BD`
- Subtle foreground: `#74808A`
- Brand blue: `#4D7CFE`
- Secure green: `#58D980`
- Warning amber: `#F5C04A`
- Blocked coral: `#F07067`
- Border: `#243241`

## Typography

- Display and labels: `Bahnschrift`
- Chinese copy: `Microsoft YaHei UI`
- Technical snippets and paths: `Cascadia Code`
- Weight contrast: display 800, body 350-400, mono 450-650.

## Motion

- Rhythm: hook-build-PEAK-proof-CTA.
- Primary transition: velocity-matched blur push, 0.45s, `power3.inOut`.
- Accent transition: block wipe using brand blue, secure green, and amber.
- Every scene has layered background texture, device/screenshot motion, entrance choreography, and one ambient motion family.

## Avoid

- No generic purple-blue gradient hero treatment.
- No centered static slide layout.
- No marketing-only claims without product UI evidence.
- No hard jump cuts between scenes.
